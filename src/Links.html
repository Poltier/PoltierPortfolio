Clip-path

Para el efecto creado en el Home sobre mis fotografías de las naturaleza, se ha utililizado la herramienta Clip-path:

La herramienta clip-path es una propiedad de CSS que se utiliza para recortar elementos de una página web. Permite definir una máscara o patrón de recorte, que se aplica a un elemento y le permite mostrar solo la parte que se encuentra dentro del patrón. La herramienta clip-path es muy útil para crear efectos visuales interesantes y personalizar la forma en la que se muestran los elementos en una página web. Por ejemplo, se puede usar para crear formas irregulares, cortar imágenes en formas específicas o crear transiciones suaves al cambiar el patrón de recorte.

Formatos de imagen

Para el desarollo de esta práctica he utilizado los siguientes formatos:

  1. AVIF es un formato de imagen que se basa en el estándar AV1 y se utiliza para comprimir imágenes de alta calidad con un tamaño de archivo reducido. Es un formato relativamente nuevo y se está volviendo cada vez más popular debido a su alta eficiencia de compresión y rendimiento en dispositivos móviles.

  2. JPEG es un formato de imagen que se utiliza a menudo para imágenes con colores y tonos degradados. Utiliza una técnica de compresión con pérdida, lo que significa que algunos detalles se pierden durante la compresión. Aunque el tamaño de archivo de las imágenes JPEG suele ser más pequeño que el de las imágenes sin comprimir, la calidad de la imagen puede verse afectada.

  3. PNG es un formato de imagen que se utiliza a menudo para imágenes con transparencias y detalles precisos. Utiliza una técnica de compresión sin pérdida, lo que significa que no se pierden detalles durante la compresión. Esto hace que el tamaño de archivo de las imágenes PNG sea generalmente más grande que el de las imágenes JPEG, pero la calidad de la imagen se mantiene mejor.
